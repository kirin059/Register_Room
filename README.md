## `Function implementation`

* Tap을 통해 '올린방' 과 '내린방' 데이터 분리하여 보여주기
* 각 방의 상세 페이지(Detail) 데이터바인딩 및 SessionStorage를 통해 데이터 수정 기능 구현 
* Form을 활용한 다양한 입력(input) 기능 구현
* 방 등록 관련한 전체 form 데이터 기입 완료 시, 방 등록 완료 모달 보여주기
